# Claude Design template — Flow Studio UI

Reference-only design source for Macro E (Flow Studio UI). This is the
Claude Design bundle handed to the program as the pixel-perfect visual
target for `laravel-flow-admin`'s UI, per
`docs/design/2026-07-07-flow-studio-ui-design-brief.md` in the core repo.

`project/` contains the design tool's own React/JSX mockup source
(`app.jsx`, `canvas.jsx`, `studio.jsx`, `styles.css`, ...) and is **not**
built or executed by this package — it is a visual/structural reference
for the real Vite + `@xyflow/react` implementation delivered across
E-PR1 through E-PR8. `project/uploads/2026-07-07-flow-studio-ui-design-brief.md`
is the design brief as originally authored against this template.

`SOURCE_README.md` is the design tool's own export README, kept verbatim
for provenance.
